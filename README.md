# Sistema de Leilões

Este projeto consiste em um sistema informatizado para uma casa de leilões, com o objetivo de auxiliar no gerenciamento de produtos, lances e usuários, trazendo mais organização, segurança e eficiência ao processo de leilão.

O sistema faz parte de um projeto acadêmico voltado à aplicação de boas práticas de versionamento de software utilizando o GitHub.

## Tecnologias Utilizadas

- Java
- MySQL
- Git
- GitHub
